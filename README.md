# Ocean Modeling using CROCO

### Coastal Dynamics | Numerical Modelling | Climate Variability

---

## Overview

This repository presents a research-oriented project on the implementation and application of the **CROCO ocean model** for the study of coastal dynamics and climate-related oceanographic processes.

The project is focused on regional hydrodynamic simulations, atmospheric and river forcing, multi-year numerical experiments, and validation against satellite and observational datasets.

---

## Research Context

Understanding coastal circulation and ocean variability is essential for improving marine environmental analysis, climate-related studies, and predictive modelling in coastal systems.

This project is framed within research activities related to:

- Physical oceanography  
- Coastal dynamics  
- Ocean–atmosphere interaction  
- Climate variability  
- Scientific computing and numerical modelling  

---

## Objectives

- Implement the CROCO regional ocean model  
- Simulate coastal circulation and oceanographic variability  
- Incorporate atmospheric and river forcing into the model setup  
- Perform multi-year numerical experiments  
- Compare outputs with satellite and observational data  

---

## Methodology

The modelling framework includes:

- Configuration of the CROCO regional model  
- Use of bathymetric and environmental forcing datasets  
- River discharge forcing for salinity representation  
- Multi-year simulations in HPC environments  
- Statistical validation against satellite products
